TUTORIAL


CONTROLES:
W: Casseb movimenta uma unidade para cima;
A: Casseb movimenta uma unidade para esquerda;
S: Casseb movimenta uma unidade para baixo;
D: Casseb movimenta uma unidade para direita;
I: Casseb interage com o objeto que está em cima.

SIMBOLOS:
&: Casseb
*: Símbolo que representa uma parede, ao se movimentar Casseb não pode passar pela parede.
@: Símbolo que representa a chave para abrir a porta para finalizar a fase, a porta abre no momento que Casseb interage com a chave.
D: Símbolo que representa a porta fechada.
=: Símbolo que representa a porta aberta quando Casseb interage com a chave.
O: Símbolo que representa um botão que o jogador pode interagir, o botão fica no chão e Cassseb deve ficar em cima dele para poder interagir.
#: Símbolo que representa um espinho. A fase é reiniciada quando Casseb toca no espinho, caso a fase seja reiniciada três vezes, o jogo volta para o menu principal.
>: Símbolo que representa um teletransporte. O teletransporte sempre deve vir em pares, quando Casseb toca em um ele é transportado para o outro e vice-versa.
? : O símbolo de interrogação é um item coringa, podendo exercer qualquer função
Obs: durante o seu percursso algumas pistas estarão escondidas pelo mapa, faça questão de achalas, pois será necesario para que casseb encontre o caminho correto
